Гид по пользованию.
В папку "input_Jpegs_here" вставьте папку с картинками.
Картинки обязаны быть названы следующем образом:
НОМЕР(Заполненый нулями - всего 6 символов), допускаются доп символы в начале.
Затем Откройте venv.exe.
{Рендер Видео
Нажмите 2. Затем введите все настройки.
Введите "0" для начала Рендера. Изначальное менеджер-окно не закроется.
В files/ВАША_ПАПКА можно добавить audio.mp3, если Вы желаете добавить звук, в ином случае окно просто не откроется
}

{Воспроизведение Видео
Нажмите 1. Затем выберите нужное видео. Наслаждайтесь
}
